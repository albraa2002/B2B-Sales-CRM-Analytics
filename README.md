 B2B Sales & CRM Analytics Dashboard

 Project Overview
A comprehensive Revenue Operations (RevOps) data analytics project. This tool simulates a 12-month B2B sales environment, tracking 2,500 deals across multiple sales representatives to identify funnel bottlenecks, optimize the sales cycle, and monitor overall pipeline health.

 Key Business Insights
Sales Rep Performance Analysis: Identified that while "Ahmed" drives the highest total revenue (EGP 19.29M), "Khaled" maintains the highest closing efficiency with a 59.8% Win Rate.
Funnel Drop-off Identification: Uncovered a critical drop-off between the 'Proposal' (447 deals) and 'Negotiation' (262 deals) stages, indicating a need for pricing or proposal strategy optimization.
Pipeline Health: Tracked an average B2B sales cycle of 50.6 days and managed an open pipeline valuation of EGP 504.51M.
UI/UX: Built a premium, corporate dark-themed interactive dashboard using HTML/CSS Grid and Plotly for an executive-level presentation.

 Tech Stack
Language: Python 3.x
Data Manipulation: Pandas, NumPy
Data Visualization: Plotly (Graph Objects), HTML/CSS
